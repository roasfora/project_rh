# Project RH - CRISP-DM Visualization

This project applies the **CRISP-DM methodology** to an HR dataset, focusing on **data analysis, visualization, and insights generation** to support decision-making in Human Resources.

## 📌 Project Overview

The main objective of this project is to explore HR-related data and identify patterns that may help understand employee behavior, workforce trends, and possible factors related to retention, performance, or organizational dynamics.

The analysis was developed in a Jupyter Notebook:

- `projeto_rh_crispdm_visualizacao.ipynb`

## 🧠 Methodology

This project is structured based on the **CRISP-DM** process:

1. **Business Understanding**  
   Define the HR problem and project objectives.

2. **Data Understanding**  
   Explore the dataset, variables, distributions, and possible issues.

3. **Data Preparation**  
   Clean the data, handle missing values, transform columns, and prepare it for analysis.

4. **Modeling / Analytical Approach**  
   Perform exploratory analysis and visual investigation of patterns.

5. **Evaluation**  
   Interpret the results and assess whether the findings answer the business questions.

6. **Deployment / Communication**  
   Present insights through clear visualizations and documentation.

## 📊 Main Contents

The notebook includes:

- Data loading and initial inspection
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Visualizations for HR indicators
- Interpretation of patterns and trends
- Conclusions based on the CRISP-DM framework
